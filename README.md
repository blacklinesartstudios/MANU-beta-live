# 💙 Manubeta Trading Assistant
> Named after Manaswi. Built with love. Lives forever.

## What's inside

### 🖥️ Windows (.exe)
`windows/manubeta_app.py` — Desktop app with:
- System tray icon (💙 in taskbar)
- Right-click: Show/Hide/Analyze/Auto Scan/Exit
- Floating widget on screen
- Reads your screen automatically
- AI trading signals

### 📱 Android (PWA)
`android/index.html` — Install as app:
- Works like a real app on Android
- Signal, Chart, Screenshot scan, Chat tabs
- Add to home screen from Chrome

### 📊 Pine Script
`manubeta_pine.pine` — TradingView indicator:
- Draws Entry, SL, T1, T2, T3 lines automatically
- Dashboard table in corner
- Buy/Sell arrows on candles

## Setup

1. Get free Groq key: console.groq.com
2. Windows: Set GROQ_KEY in manubeta_app.py → run BUILD_EXE.bat
3. Android: Host android/ on GitHub Pages → open on phone → Add to Home Screen
4. TradingView: Paste Pine Script in editor → Add to chart
